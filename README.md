arXiv Haiku
=====

Accidental haiku, auto-generated from the arXiv from http://arxiv.org/list/astro-ph/new

### Usage

1. Install dependencies (see top of `runner.rb`)
2. `ruby runner.rb`
3. Enjoy accidental haiku
